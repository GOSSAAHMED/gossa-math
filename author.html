About the author & the framework

Ahmed Gossa develops a mathematical framework called Hierarchical Calculus. The framework treats certain constructions at multiple hierarchical levels (orders), rather than restricting everything to a single classical level. Within this site: D0 denotes the classical differential derivative, and D1 denotes the relative derivative. The general symbol \(\,D_n^1\,\) denotes the hierarchical derivative of order \(n\) and degree \(1\).

Why this page matters Search engines and readers both reward consistency: one definition, one notation, and clear internal linking. That is why the unified definition below is treated as the single source of truth across all pages.

Unified definition of the hierarchical derivative \(D_n^1\)

The site adopts the following definition (and only this definition):

\[ \boxed{ D_n^{1} f(x) = \frac{d\,\ln^{(n)}\!\big(f(x)\big)} {d\,\ln^{(n)}\!(x)} } \]

Here, \(\ln^{(n)}\) denotes the iterated logarithm of order \(n\) (apply the natural logarithm \(n\) times), and \(n \in \mathbb{N}\). This definition is transform-symmetric because the same hierarchical transform is applied to both the variable \(x\) and the function value \(f(x)\).

Consistency note (very important) Throughout this site, we do not use mixed forms such as \(\frac{d(\ln f)}{d\ln^{(n)}x}\). The adopted definition always applies \(\ln^{(n)}\) on both sides: \(\ln^{(n)}(f(x))\) versus \(\ln^{(n)}(x)\). This prevents contradictions between pages and ensures all rules are derived from one origin.

Short sanity-check examples (D0 and D1)

The following examples are intentionally short. They serve as quick verification checks before the reader moves to the full derivations and rule pages.

Example 1: Exponential function — \(f(x)=e^x\)

For \(n=0\) (classical derivative):

\[ D_0^1(e^x)=\frac{d(e^x)}{dx}=e^x \]

For \(n=1\) (relative derivative \(D1\)):

\[ D_1^1(e^x) = \frac{d\,\ln(e^x)}{d\,\ln(x)} = \frac{d(x)}{d(\ln x)} = \frac{dx}{d(\ln x)} = x \]

Example 2: Power function — \(f(x)=x^a\) with \(x>0\)

\[ D_1^1(x^a) = \frac{d\,\ln(x^a)}{d\,\ln(x)} = \frac{d(a\ln x)}{d(\ln x)} = a \]

Example 3: Product — \(f(x)=x^a\,g(x)\) with \(x>0\) and \(g(x)>0\)

Since \(\ln(x^a g)=\ln(x^a)+\ln(g)\), we obtain:

\[ D_1^1(x^a g(x)) = D_1^1(x^a) + D_1^1(g(x)) = a + D_1^1(g(x)) \]

For full derivations and the complete chain/product/quotient rules under the unified definition, see Rules.

Consistency rules (to prevent contradictions)

  • Notation: we always write \(D_n^1\) with explicit order \(n\) and degree \(1\). A symbol like \(D_n f\) is considered incomplete within this site.
  • Domain: because \(\ln^{(n)}\) appears, the expressions entering the iterated logarithm must be positive on the studied domain (e.g., \(x>0\) and \(f(x)>0\) at the relevant order).
  • Rule source: every rule on this site is derived from the unified definition shown above—no alternative definition is used.
