19.1 Core Idea

  • If \(\widehat{D_n}\) is (nearly) constant â†’ infer a closed-form law for \(y(x)\).
  • If \(\widehat{D_n}\) is (nearly) linear in \(L_n(x)\) â†’ infer a standard linear model at level \(n\).
  • If nothing stabilizes â†’ the phenomenon may require rescaling or a different model family.

19.2 Ready-to-Use Inference Rules

(A) If \(D_1y=a\) constant â†’ \(y=Cx^a\).
(B) If \(D_2y=a\) constant â†’ \(y=\exp(A(\ln x)^a)\).
(C) If \(D_3y=a\) constant â†’ \(y=\exp(\exp(A(\ln\ln x)^a))\).

19.3 Fitting Constants from Data

  • For \(y=Cx^a\): estimate \(a\) from \(\widehat{D_1}\), then \(C=y/x^a\).
  • For \(y=\exp(A(\ln x)^a)\): estimate \(a\) from stable \(\widehat{D_2}\), then \(A=\ln y/(\ln x)^a\).
  • For \(y=\exp(\exp(A(\ln\ln x)^a))\): \(A=\ln\ln y/(\ln\ln x)^a\).

19.4 Linear (Not Constant) Patterns

If \(\widehat{D_1}\approx \alpha+\beta\ln x\), then \[ y=Cx^\alpha\exp\!\left(\frac{\beta}{2}(\ln x)^2\right). \]
