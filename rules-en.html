Hierarchical Derivation Rules

This page presents the core rules of hierarchical differentiation. We begin with the relative derivative and then generalize to rank \(n\).
Notation rule: every derivative is written with explicit rank and degree: \(\;D_r^n\). In this page we explicitly use degree 1: \(\;D_{1}^{1}\) and \(\;D_{n}^{1}\).

Concept DOI: 10.5281/zenodo.17917302
Version DOI (v1.0.0): 10.5281/zenodo.17917303

1) Quick definition: the relative derivative (rank 1, degree 1)

Assume \(f(x)>0\) on a domain \(x>0\). The relative derivative measures change on a relative scale.

\[ \boxed{ D_{1}^{1} f(x) =\frac{d}{d\ln x}\big(\ln f(x)\big) =\frac{d\ln f(x)}{d\ln x} } \]
Idea: If \(D_{1}^{1} f\) is constant, then \(f\) behaves like a power \(x^\alpha\) (scaling law).

2) Core rules for \(D_{1}^{1}\)

2.1) Multiplication by a constant

\[ \boxed{ D_{1}^{1}(C\,f)=D_{1}^{1} f \qquad (C>0\ \text{constant}) } \]

2.2) Product rule (relative composition)

\[ \boxed{ D_{1}^{1}(fg)=D_{1}^{1} f + D_{1}^{1} g } \]

2.3) Quotient rule

\[ \boxed{ D_{1}^{1}\!\left(\frac{f}{g}\right)=D_{1}^{1} f - D_{1}^{1} g } \]

2.4) Constant exponent

\[ \boxed{ D_{1}^{1}\!\left(f^a\right)=a\,D_{1}^{1} f \qquad (a\ \text{constant}) } \]

2.5) Power function

\[ \boxed{ D_{1}^{1}(x^\alpha)=\alpha } \]

2.6) Chain rule — relative version

Let \(u=u(x)>0\) and \(f=f(u)>0\). Then:

\[ \boxed{ D_{1}^{1}\big(f(u(x))\big)=\Big(D_{1}^{1} f\Big)(u)\;\cdot\;D_{1}^{1}u(x) } \]

3) Quick checks for \(D_{1}^{1}\)

Function \(f(x)\) Result Comment
\(f(x)=x^3\) \(D_{1}^{1} f=3\) explicit relative homogeneity
\(f(x)=\sqrt{x}=x^{1/2}\) \(D_{1}^{1} f=\tfrac{1}{2}\) power law
\(f(x)=e^x\) \(D_{1}^{1} f=x\) \(\ln f=x\), then differentiate w.r.t. \(\ln x\)
\(f(x)=\ln x\) \(D_{1}^{1} f=\dfrac{1}{\ln x}\) differentiate \(\ln(\ln x)\) w.r.t. \(\ln x\)

4) Generalization to rank \(n\): \(D_{n}^{1}\)

The hierarchical principle: going to a higher rank means changing the “measurement scale” of variation.

\[ \boxed{ D_{n}^{1} f(x)=\frac{d}{d\ln^{(n)}x}\big(\ln f(x)\big) } \]

With this definition, the same structural rules repeat at every rank: the logarithm turns the rank operation into addition, and differentiation yields a clean rule.

\[ \boxed{ D_{n}^{1}(fg)=D_{n}^{1} f + D_{n}^{1} g,\qquad D_{n}^{1}\!\left(\frac{f}{g}\right)=D_{n}^{1} f - D_{n}^{1} g,\qquad D_{n}^{1}(f^a)=a\,D_{n}^{1} f } \]
Hierarchical interpretation: Rank 0 composition is additive, rank 1 is multiplicative, and higher ranks follow their own scale through \(\ln^{(n)}\).
