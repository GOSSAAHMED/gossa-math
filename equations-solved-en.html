1) The hierarchical lifting principle

Many “hard” classical ODEs become easier when expressed on the correct scale: relative scale (\(D_1\)) or logarithmic scale (\(D_2\)).

\[ \boxed{ D_1 y=\frac{d\ln y}{d\ln x}=\frac{x}{y}\frac{dy}{dx} }\qquad (x>0,\ y>0) \]
\[ \boxed{ D_2 y=\frac{d\ln y}{d\ln\ln x} =\frac{x\ln x}{y}\frac{dy}{dx} }\qquad (x>1,\ y>0) \]
Golden rule:
If \(\displaystyle y'=\frac{y}{x}F(x,y)\), then \(\displaystyle D_1 y=F(x,y)\).
If \(\displaystyle y'=\frac{y}{x\ln x}G(x,y)\), then \(\displaystyle D_2 y=G(x,y)\).

2) Six fully solved examples

Example 1 (direct \(D_1\)): \(\;y'=\dfrac{a}{x}y\)

\[ \frac{dy}{dx}=\frac{a}{x}y \Rightarrow \frac{x}{y}\frac{dy}{dx}=a \Rightarrow \boxed{D_1 y=a} \]
\[ \frac{d\ln y}{d\ln x}=a \Rightarrow \ln y=a\ln x + C \Rightarrow \boxed{y=C_\times x^a} \]

Example 2 (simpler in \(D_1\)): \(\;y'=\dfrac{y}{x}\ln x\)

\[ \frac{dy}{dx}=\frac{y}{x}\ln x \Rightarrow \boxed{D_1 y=\ln x} \]

Let \(t=\ln x\). Then \(\dfrac{d\ln y}{dt}=t\).

\[ \ln y=\frac{t^2}{2}+C \Rightarrow \boxed{y=C_\times\exp\!\left(\frac{(\ln x)^2}{2}\right)} \]

Example 3 (direct \(D_2\)): \(\;y'=\dfrac{a}{x\ln x}y\)

\[ \frac{dy}{dx}=\frac{a}{x\ln x}y \Rightarrow \frac{x\ln x}{y}\frac{dy}{dx}=a \Rightarrow \boxed{D_2 y=a} \]
\[ \frac{d\ln y}{d\ln\ln x}=a \Rightarrow \ln y=a\ln\ln x + C \Rightarrow \boxed{y=C_\times(\ln x)^a} \]

Example 4 (relative integration): \(\;y'=\dfrac{y}{x}(a+b\ln x)\)

\[ \frac{dy}{dx}=\frac{y}{x}(a+b\ln x)\Rightarrow \boxed{D_1 y=a+b\ln x} \]

Let \(t=\ln x\). Then \(\dfrac{d\ln y}{dt}=a+bt\).

\[ \ln y=at+\frac{b}{2}t^2 + C \Rightarrow \boxed{ y=C_\times\,x^a\,\exp\!\left(\frac{b}{2}(\ln x)^2\right) } \]

Example 5 (nonlinear, separable in scale): \(\;y'=\dfrac{1}{x}y^m\)

\[ \frac{dy}{dx}=\frac{1}{x}y^m \Rightarrow \frac{x}{y}\frac{dy}{dx}=y^{m-1} \Rightarrow \boxed{D_1 y=y^{m-1}} \]

Using \(d\ln y=\frac{dy}{y}\):

\[ \frac{d\ln y}{d\ln x}=y^{m-1} \Rightarrow \frac{dy}{y^m}=d\ln x \Rightarrow \int y^{-m}dy=\int d\ln x \]
\[ \frac{y^{1-m}}{1-m}=\ln x + C \Rightarrow \boxed{ y(x)=\left((1-m)(\ln x + C)\right)^{\frac{1}{1-m}} }\ (m\neq1,\ x>0) \]

Example 6 (natural \(D_2\)): \(\;y'=\dfrac{y}{x\ln x}\ln\ln x\)

\[ \frac{dy}{dx}=\frac{y}{x\ln x}\ln\ln x \Rightarrow \boxed{D_2 y=\ln\ln x} \]

Let \(u=\ln\ln x\). Then \(\dfrac{d\ln y}{du}=u\).

\[ \ln y=\frac{u^2}{2}+C \Rightarrow \boxed{ y=C_\times\exp\!\left(\frac{(\ln\ln x)^2}{2}\right) } \]
Takeaway: If an ODE is naturally written as \(\frac{y}{x}\times(\cdots)\), \(D_1\) is the correct scale. If it is \(\frac{y}{x\ln x}\times(\cdots)\), \(D_2\) is the correct scale. The “difficulty” is often a scale-choice artifact.
