قوانين الاشتقاق الهرمي

في هذه الصفحة نعرض القواعد الأساسية للاشتقاق الهرمي. نبدأ بالمشتق النسبي ثم نُعمِّم إلى الرتبة \(n\).
قاعدة الترميز: نكتب أي مشتق بصيغة الرتبة والدرجة: \(\;D_r^n\). وفي هذه الصفحة نستعمل الدرجة \(1\) صراحة: \(\;D_{1}^{1}\) و \(\;D_{n}^{1}\).

Concept DOI: 10.5281/zenodo.17917302
Version DOI (v1.0.0): 10.5281/zenodo.17917303

1) تعريف سريع للمشتق النسبي (رتبة 1، درجة 1)

نعتبر دالة موجبة \(f(x)>0\) على مجال \(x>0\). المشتق النسبي يقيس التغير بالنسبة لمقياس نسبي.

\[ \boxed{ D_{1}^{1} f(x) =\frac{d}{d\ln x}\big(\ln f(x)\big) =\frac{d\ln f(x)}{d\ln x} } \]
فكرة: إذا كان \(D_{1}^{1} f\) ثابتًا، فهذا يعني أن \(f\) تتصرف كقوة \(x^\alpha\) (قانون تحجيم).

2) القواعد الأساسية لـ \(D_{1}^{1}\)

2.1) ثابت مضروب

\[ \boxed{ D_{1}^{1}(C\,f)=D_{1}^{1} f \qquad (C>0\ \text{ثابت}) } \]

لأن \(\ln(Cf)=\ln C+\ln f\) واشتقاق \(\ln C\) بالنسبة لـ \(\ln x\) يساوي صفرًا.

2.2) قاعدة الضرب (التراكب النسبي)

\[ \boxed{ D_{1}^{1}(fg)=D_{1}^{1} f + D_{1}^{1} g } \]

2.3) قاعدة القسمة

\[ \boxed{ D_{1}^{1}\!\left(\frac{f}{g}\right)=D_{1}^{1} f - D_{1}^{1} g } \]

2.4) ثابت أُسّي (رفع لقوة ثابتة)

\[ \boxed{ D_{1}^{1}\!\left(f^a\right)=a\,D_{1}^{1} f \qquad (a\ \text{ثابت}) } \]

2.5) دالة قوة: \(x^\alpha\)

\[ \boxed{ D_{1}^{1}(x^\alpha)=\alpha } \]

2.6) قاعدة السلسلة — النسخة النسبية

لتكن \(u=u(x)\) موجبة و\(f=f(u)\) موجبة. عندئذٍ:

\[ \boxed{ D_{1}^{1}\big(f(u(x))\big)=\Big(D_{1}^{1} f\Big)(u)\;\cdot\;D_{1}^{1}u(x) } \]

المعنى: “معدل التغير النسبي للمركب” يساوي حاصل ضرب المعدلين النسبيين.

3) أمثلة تحقق سريعة لـ \(D_{1}^{1}\)

الدالة \(f(x)\) النتيجة تعليق
\(f(x)=x^3\) \(D_{1}^{1} f=3\) تجانس نسبي صريح
\(f(x)=\sqrt{x}=x^{1/2}\) \(D_{1}^{1} f=\tfrac{1}{2}\) قانون قوة
\(f(x)=e^x\) \(D_{1}^{1} f=x\) \(\ln f=x\) ثم الاشتقاق بالنسبة لـ \(\ln x\)
\(f(x)=\ln x\) \(D_{1}^{1} f=\dfrac{1}{\ln x}\) \(\ln(\ln x)\) مشتقتها بالنسبة لـ \(\ln x\) هي \(1/\ln x\)

4) التعميم إلى الرتبة \(n\): \(D_{n}^{1}\)

المبدأ الهرمي العام: عند الانتقال من رتبة إلى رتبة أعلى نستبدل “مقياس التغير”.

\[ \boxed{ D_{n}^{1} f(x)=\frac{d}{d\ln^{(n)}x}\big(\ln f(x)\big) } \]

ومع هذا التعريف، تتكرر القواعد البنيوية نفسها على كل رتبة (بنفس الفكرة): اللوغاريتم يحوّل “عملية الرتبة” إلى جمع، ثم الاشتقاق يعيدها إلى قانون بسيط.

\[ \boxed{ D_{n}^{1}(fg)=D_{n}^{1} f + D_{n}^{1} g,\qquad D_{n}^{1}\!\left(\frac{f}{g}\right)=D_{n}^{1} f - D_{n}^{1} g,\qquad D_{n}^{1}(f^a)=a\,D_{n}^{1} f } \]
تفسير هرمي مختصر: في الرتبة 0 التراكب جمعي، في الرتبة 1 التراكب ضربي، وفي الرتب الأعلى يصبح التراكب هرميًا وفق مقياس الرتبة.
