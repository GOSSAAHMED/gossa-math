قوانين الاشتقاق الهرمي

في هذه الصفحة نعرض القواعد الأساسية للاشتقاق الهرمي. نبدأ بالمشتق النسبي \(D_1\)، ثم نُعمِّم إلى \(D_n\).

هذه القوانين جزء من الإطار التأسيسي للحساب الهرمي. أي استعمال أو توسعة يجب أن تُرفق بالاستشهاد بـ Zenodo وبالموقع الرسمي.

1) تعريف سريع للمشتق النسبي \(D_1\)

نعتبر دالة موجبة \(f(x)>0\) على مجال \(x>0\). المشتق النسبي يقيس التغير بالنسبة لمقياس نسبي.

\[ \boxed{ D_1 f(x)=\frac{d}{d\ln x}\big(\ln f(x)\big)=\frac{d\ln f(x)}{d\ln x} } \]
فكرة: إذا كانت \(D_1 f\) ثابتًا، فهذا يعني أن \(f\) تتصرف كقوة \(x^\alpha\) (قانون تحجيم).

2) القواعد الأساسية لـ \(D_1\)

2.1) ثابت مضروب

\[ \boxed{ D_1(C\,f)=D_1 f \qquad (C>0\ \text{ثابت}) } \]

لأن \(\ln(Cf)=\ln C+\ln f\) واشتقاق \(\ln C\) يساوي صفرًا.

2.2) قاعدة الضرب (التراكب النسبي)

\[ \boxed{ D_1(fg)=D_1 f + D_1 g } \]

برهان مختصر: \(\ln(fg)=\ln f+\ln g\) ثم الاشتقاق بالنسبة لـ \(\ln x\).

2.3) قاعدة القسمة

\[ \boxed{ D_1\!\left(\frac{f}{g}\right)=D_1 f - D_1 g } \]

لأن \(\ln(f/g)=\ln f-\ln g\).

2.4) ثابت أُسّي (رفع لقوة ثابتة)

\[ \boxed{ D_1\!\left(f^a\right)=a\,D_1 f \qquad (a\ \text{ثابت}) } \]

لأن \(\ln(f^a)=a\ln f\).

2.5) دالة قوة: \(x^\alpha\)

\[ \boxed{ D_1(x^\alpha)=\alpha } \]

لأن \(\ln(x^\alpha)=\alpha\ln x\) واشتقاقها بالنسبة لـ \(\ln x\) يساوي \(\alpha\).

2.6) قاعدة السلسلة (Chain Rule) — النسخة النسبية

لتكن \(u=u(x)\) موجبة و\(f=f(u)\) موجبة. عندئذٍ:

\[ \boxed{ D_1\big(f(u(x))\big)=\Big(D_1 f\Big)(u)\;\cdot\;D_1 u(x) } \]
المعنى: “معدل التغير النسبي للمركب” يساوي حاصل ضرب المعدلين النسبيين.

3) أمثلة تحقق سريعة لـ \(D_1\)

الدالة \(f(x)\) النتيجة تعليق
\(f(x)=x^3\) \(D_1 f=3\) تجانس نسبي صريح
\(f(x)=\sqrt{x}=x^{1/2}\) \(D_1 f=\tfrac{1}{2}\) قانون قوة
\(f(x)=e^x\) \(D_1 f=x\) \(\ln f=x\) ثم الاشتقاق بالنسبة لـ \(\ln x\)
\(f(x)=\ln x\) \(D_1 f=\frac{1}{\ln x}\) لأن \(\ln(\ln x)\) مشتقتها بالنسبة لـ \(\ln x\) هي \(1/\ln x\)

4) التعميم إلى \(D_n\)

المبدأ الهرمي العام: عند الانتقال من رتبة إلى رتبة أعلى نستبدل مقياس التغير.

\[ \boxed{ D_n f(x)=\frac{d}{d\ln^{(n)}x}\big(\ln f(x)\big) } \]

ومع هذا التعريف، تتكرر القواعد البنيوية نفسها على كل رتبة (مع نفس الفكرة): \(\ln\) يحوّل عمليات “الرتبة” إلى جمع، ثم الاشتقاق يعيدها إلى قانون بسيط.

\[ \boxed{ D_n(fg)=D_n f + D_n g,\qquad D_n\!\left(\frac{f}{g}\right)=D_n f - D_n g,\qquad D_n(f^a)=a\,D_n f } \]
تراكب هرمي (فكرة أساسية): في الرتبة 0 التراكب جمعي، في الرتبة 1 التراكب ضربي، وفي الرتب الأعلى يصبح التراكب هرميًا وفق مقياس الرتبة.
